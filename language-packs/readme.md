# Uniview Language Customization 

Uniview supports multiple languages, out of box: 

* English, 
* French, 
* Spanish  
* Chinese. 

Other Language will be provided on demand.

The current repo provides clients an entry for the possible need of further customization, such as adding a new language set, client self-editing to their own flavor. The final package will be packed into the build for deliverying.
